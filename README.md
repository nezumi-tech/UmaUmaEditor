# UmaUmaUpdater
個人で更新したUmamusumeLibrary.jsonをダウンロードしてUmaUmaCruiseを起動するアプリです。  
UmaUmaCruiseの更新が止まったため、制作しました。

## 使い方
・Releaseからzipファイルをダウンロード  
・解凍したものをフォルダごとUmaUmaCruise.exeがあるフォルダに配置  
・UmaUmaUpdaterを起動
